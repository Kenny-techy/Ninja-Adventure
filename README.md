# Ninja-Adventure
Ninja Adventure is a 2D platformer developed in Unity using C# as part of my BTEC Computing Game Development assignment. In the game the player controls a ninja who must complete timed levels by avoiding hazards, defeat enemies, collect coins, and unlocking doors to progress. Each level increases in difficulty through additional hazards, enemies, and reduced completion time.
